# Build-Scripts #

This is a bunch of scripts to build third-party libraries for my projects.

Required tools:

- Visual Studio 2015
- 7-zip
- MSYS installation
- Ninja
- JOM
- CMake
- NASM (Netwide Assembler)
- Perl (ActiveState Perl)
